# Femboy Flashbang
A basic flashbang/jumpscare animation using Tkinter. Feel free to modify this code for any use.

# Run
To run, simply clone the repository, run<br>
<code>pip install pygame</code>, then<br>
<code>python fmb.py</code>.<br><br>
If <code>pip install pygame</code> fails, make sure you are on Python 3.13 and before.
